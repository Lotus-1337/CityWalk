# CityWalk

## Unreal Engine Project which focuses on AI development and optimisation. 

It's A Game demo focused heavily on the code and not the visual side of things.
It Includes a Custom written A* Algorithm, Building Portals 
and Funnel Algorithm for finding the most optimal path as fast as possible
Another Big Part of this Project is the amount of AI's it can run 
withour larger performance drops.
